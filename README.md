# ci-cd-templates

Reusable CI/CD templates and configurations for infrastructure and application deployments.

## Repository Structure

- `ansible/` - Ansible playbooks and configurations
- `helm-charts/` - Helm chart values and configurations
- `scripts/` - Utility scripts for various services
- `secrets/` - Secret configuration documentation

## Secrets Configuration

All secret configuration documentation is organized in the `secrets/` directory. Refer to [SECRETS_SETUP.md](./SECRETS_SETUP.md) for an overview of available secret configurations.

## Usage

This repository provides reusable templates for:
- Infrastructure provisioning and configuration
- Application deployments
- Observability stack setup
- CI/CD pipeline configurations

Import workflows and playbooks as needed for your projects.
